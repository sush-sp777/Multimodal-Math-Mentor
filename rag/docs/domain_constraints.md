# Domain Constraints & Validity Checks

## Square Roots
Expression inside √ must be ≥ 0

## Logarithms
Argument of log must be > 0

## Denominators
Denominator ≠ 0

## Probability
Final probability must lie between 0 and 1

## Trigonometric
sin x, cos x ∈ [-1, 1]

## Verifier Checklist
- Units consistent
- Domain satisfied
- Final answer simplified
