# Linear Algebra – Basics

## Matrices
Order: m × n

## Matrix Operations
- Addition: same order required
- Multiplication: (m×n) × (n×p)

## Determinant (2×2)
|a b|
|c d| = ad - bc

## Inverse of 2×2 Matrix
A⁻¹ = (1 / det A) ×
| d  -b |
| -c  a |

Valid only if det(A) ≠ 0

## System of Linear Equations
- Unique solution: det ≠ 0
- Infinite or no solution: det = 0

## Common Mistakes
- Multiplication order confusion
- Forgetting determinant condition
