# Problem Solving Templates (Reusable Patterns)

## Template 1: Algebraic Equation
1. Write given information clearly
2. Convert to equation
3. Simplify
4. Solve
5. Check solution in original equation

## Template 2: Probability
1. Define sample space
2. Count total outcomes
3. Count favorable outcomes
4. Apply formula
5. Simplify result

## Template 3: Calculus Optimization
1. Define variable
2. Write function
3. Find derivative
4. Set derivative = 0
5. Verify max/min
